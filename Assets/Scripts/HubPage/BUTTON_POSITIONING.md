# ボタンの位置調整ガイド

## 📍 ボタンの位置を動かす方法

HubPageManager には**座標指定**と**レイアウトモード**でボタンの位置を調整できます。

---

## 🎯 方法1: レイアウトモードで自動配置（推奨）
- ボタンが縦に並びます
- **ボタン間のスペース** で間隔を調整
- **開始位置** で最初のボタンの位置を設定

#### ✅ Horizontal（横方向）
- ボタンが横に並びます
- **ボタン間のスペース** で間隔を調整
- **開始位置** で最初のボタンの位置を設定

#### ✅ Grid（グリッド）
- ボタンがグリッド状に並びます
- **1行あたりのボタン数** で列数を設定（例: 3）
- **グリッドセルサイズ** でマス目のサイズを設定
- **開始位置** でグリッド全体の位置を調整

#### Custom（カスタム）
- 各ボタンの個別設定を使用
- 詳しくは「方法2」を参照

---

## 🎨 方法2: 個別のボタンを自由に配置

### 手順
1. HubPageManager を選択
2. ボタンリストから編集したいボタンを展開
3. **位置設定** セクションを開く
4. **「カスタム位置を使用」** をチェック

### 設定項目

#### アンカー
ボタンの基準点を設定します:
- **TopLeft**: 画面左上を基準
- **TopCenter**: 画面上部中央を基準
- **TopRight**: 画面右上を基準
- **MiddleLeft**: 画面左中央を基準
- **Center**: 画面中央を基準（デフォルト）
- **MiddleRight**: 画面右中央を基準
- **BottomLeft**: 画面左下を基準
- **BottomCenter**: 画面下部中央を基準
- **BottomRight**: 画面右下を基準
- その他のStretchオプション: 画面サイズに合わせて伸縮

#### 位置 (X, Y)
アンカーからの相対位置:
- **X**: 横方向の位置（正の数で右、負の数で左）
- **Y**: 縦方向の位置（正の数で上、負の数で下）

### 例

#### 画面左上に配置
```
アンカー: TopLeft
位置: X=20, Y=-20
```

#### 画面中央に配置
```
アンカー: Center
位置: X=0, Y=0
```

#### 画面右下に配置
```
アンカー: BottomRight
位置: X=-20, Y=20
```

---

## 🤔 どの方法を使えばいい？

### 方法1がおすすめな場合：
- ✅ 複数のボタンを規則的に配置したい
- ✅ 自動で整列させたい
- ✅ レイアウトパターンを再利用したい
- ✅ 均等に並べたい

### 方法2がおすすめな場合：
- ✅ 正確な座標が必要
- ✅ 各ボタンを個別に配置したい
- ✅ 数値で管理したい
- ✅ 自由な配置が必要

**💡 最初は方法1で大まかに配置して、方法2で個別に微調整するのが効率的！**

## 💡 実用例

### 例1: メニューボタンを縦に並べる
```
レイアウトモード: Vertical
ボタン間のスペース: 15
開始位置: X=0, Y=-50
```

### 例2: 横3列のグリッド
```
レイアウトモード: Grid
1行あたりのボタン数: 3
グリッドセルサイズ: 320 x 70
開始位置: X=0, Y=-100
```

### 例3: 特定のボタンだけ別の位置に
```
レイアウトモード: Vertical（他のボタン用）

特定のボタン:
  カスタム位置を使用: ✓
  アンカー: BottomRight
  位置: X=-30, Y=30
```

## 🔄 プレビューと調整

### リアルタイムプレビュー
1. 設定を変更
2. **「ボタンをプレビュー」** をクリック
3. Scene ビューまたは Game ビューで確認
4. 満足いくまで調整を繰り返す

### Play モードで確認
1. Play ボタンを押す
2. 実際の動作を確認
3. 必要に応じて微調整

---

## ⚙️ 高度な使い方

### Button Container の位置を変更
Button Container 自体を移動させることで、すべてのボタンをまとめて移動できます:

1. Hierarchy で **ButtonContainer** を選択
2. Inspector の **RectTransform** で位置を調整
3. または Scene ビューでドラッグ

### 開始位置の活用
レイアウトモードが Vertical/Horizontal/Grid の場合:
- **開始位置** でボタングループ全体の位置を調整
- 個別のボタン間隔は **ボタン間のスペース** で調整

## 🐛 トラブルシューティング

### Q: ボタンの位置が変わらない
**A**: 以下を確認:
1. 設定変更後に「ボタンをプレビュー」をクリックしたか
2. カスタム位置を使用している場合、チェックが入っているか
3. レイアウトモードが意図した設定になっているか

### Q: ボタンが画面外に出てしまう
**A**: 
1. アンカー設定を確認
2. 位置の値を小さくする
3. Button Container の位置を確認

### Q: グリッドがうまく並ばない
**A**:
1. **1行あたりのボタン数** を調整
2. **グリッドセルサイズ** をボタンサイズより大きくする
3. **開始位置** で全体の位置を調整

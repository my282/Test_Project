# クイックスタートガイド - シーン遷移ボタンシステム

## 📋 3分でできる！完全自動セットアップ

### ステップ1: Canvas を作成（まだない場合）
1. Hierarchy で右クリック → **UI** → **Canvas**

### ステップ2: セットアップウィザードを起動
1. Unity メニューバーから **Tools** → **シーン遷移ボタン** → **セットアップウィザード** を選択

### ステップ3: 自動セットアップ
1. ウィザードで以下を設定:
   - ✓ サンプルボタンを作成: チェック
   - ボタン数: 3（お好みで変更可能）
   - ✓ サンプルシーンも作成: **チェック**（重要！）
   - ✓ ビルド設定に追加: **チェック**（重要！）
2. **「セットアップを実行」** ボタンをクリック

### ステップ4: 確認
1. Scene ビューまたは Game ビューで**ボタンが既に表示されています！**（Play 不要）
2. **Play ボタン** を押してゲームを実行
3. ボタンをクリックしてシーン遷移を確認

🎉 **完了！** すぐに動作するシーン遷移ボタンが完成しました！

## 🎨 ボタンの位置を調整する

### 🖱️ 超簡単！Scene ビューでドラッグ（推奨）
1. HubPageManager Inspector を開く
2. **「✎ Scene ビューで位置調整」** をクリック
3. Scene ビューでボタンをドラッグして移動
4. **Ctrl を押しながら**でグリッドにスナップ
5. 完了したら **「■ 位置調整モード終了」**

**座標を入力する必要なし！マウスで自由に動かせます！**

### ⌨️ 数値で精密調整（オプション）

#### 自動配置モード
1. HubPageManager Inspector を開く
2. **レイアウト設定** を展開
3. **レイアウトモード** を選択:
   - **Vertical**: 縦に並べる
   - **Horizontal**: 横に並べる  
   - **Grid**: グリッド状に並べる
   - **Custom**: 個別に配置
4. **ボタン間のスペース** で間隔を調整
5. **開始位置** で全体の位置を調整

#### 個別配置
1. ボタンを展開 → **位置設定**
2. **「カスタム位置を使用」** をチェック
3. **アンカー** で基準点を選択（例: TopLeft, Center, BottomRight）
4. **位置 (X, Y)** で座標を設定

**詳しくは**: [BUTTON_POSITIONING.md](BUTTON_POSITIONING.md) を参照

## 🎨 画像を使ったボタンを作る（追加5分）

### 画像の準備
1. ウィザードで **「サンプル画像フォルダを作成」** をクリック
2. `Assets/Resources/ButtonImages` フォルダが作成されます
3. このフォルダに画像ファイルをドラッグ&ドロップ

### 画像をスプライトに変換
1. インポートした画像を選択
2. Inspector で以下を設定:
   ```
   Texture Type: Sprite (2D and UI)
   ```
3. **Apply** をクリック

### ボタンに画像を適用
1. HubPageManager の Inspector を開く
2. ボタンを展開 → **画像設定**
3. **背景画像** に画像をドラッグ&ドロップ
4. Play で確認！

## 🔧 トラブルシューティング

### Q: ボタンが表示されない
**A**: 以下を確認してください
- ✓ Canvas が存在するか
- ✓ Button Container が設定されているか  
- ✓ HubPageManager の「ボタンをプレビュー」を押してみる

### Q: ボタンをクリックしてもシーン遷移しない
**A**: 自動セットアップを使用した場合、この問題は発生しません！
手動で設定した場合は：
1. HubPageManager の Inspector で **「シーン検証」** ボタンをクリック
2. 問題があれば詳細がコンソールに表示されます
3. シーンが見つからない場合:
   - セットアップウィザードの「サンプルシーンを作成」を使用
   - または File → New Scene で手動作成
4. ビルド設定に未追加の場合:
   - File → Build Settings を開く
   - シーンファイルをドラッグ&ドロップ

### Q: シーン遷移時にエラーが出る
**A**: コンソールのエラーメッセージを確認してください
- シーン名のスペルミス（大文字小文字を含む）
- シーンがビルド設定に追加されていない
- 詳細なエラー情報がコンソールに表示されます

### Q: 画像が表示されない
**A**: 画像設定を確認してください
- ✓ Texture Type が「Sprite (2D and UI)」になっているか
- ✓ Apply を押したか
- ✓ 背景色が白になっているか（画像のティント）

## 💡 便利な機能


### 🎯 シーン検証機能
HubPageManager Inspector の「シーン検証」ボタンで：
- すべてのシーンが存在するか確認
- ビルド設定への追加状態を確認
- 問題があれば詳細な解決方法を表示

### 🔍 デバッグ機能
HubPageManager を右クリック：
- **「ボタンを再生成」**: ボタンを即座に更新
- **「ビルド設定のシーン一覧を表示」**: 現在のビルド設定を確認

### 📦 自動シーン作成
セットアップウィザードの機能：
- サンプルシーンを自動作成
- 各シーンに戻るボタンを自動配置
- ビルド設定に自動追加

### カスタムエディタの機能
HubPageManager を選択すると、専用のカスタムエディタが表示されます:
- ボタンの追加・削除が簡単
- 折りたたみ可能なセクション
- 「ボタンをプレビュー」で即座に反映（Play 不要！）
- 「シーン検証」で問題を即座にチェック

### コンテキストメニュー
HubPageManager を右クリック → **「ボタンを再生成」** で、エディタモードでもボタンを即座に更新できます。

### メニューからのアクセス
Unity メニュー → **Tools** → **シーン遷移ボタン** から各種ツールにアクセスできます。
